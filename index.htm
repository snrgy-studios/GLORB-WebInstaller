Welcome to the GLORB Web Installer!

  1. Select firmware version and hardware options (probably defaults)
  2. Click Install to display the COM port list in a popup
  3. Connect your GLORB via USB-C and select the port that appears in the list
  4. Click Install GLORB and wait for the installation to finish
  5. (Optional) Enter your WiFi details
  6. Power up your GLORB and control it through the app!
