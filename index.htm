Welcome to the GLORB Installer!

Installation Steps

  1. Select firmware version (probably default)
  2. Click SELECT DEVICE to display the COM port list in a popup
  3. Connect your GLORB via USB-C and select the device that appears in the list
  4. Click Install GLORB and wait for the installation to finish
  5. Configure your WiFi
  6. Press Visit Device or control it through the app!
